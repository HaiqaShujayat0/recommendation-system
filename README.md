# GLYERAL UI

Medical dashboard UI for physician decision support (frontend-only, dummy data).

## Quick start

```bash
cd glyeral-ui
npm install
npm run dev
```

Open the URL shown in the terminal (e.g. `http://localhost:5173`).

## Docs

See **[SETUP_GUIDE.md](./SETUP_GUIDE.md)** for:

## Stack

- **React 18** + **Vite**
- **Tailwind CSS**
- **lucide-react** (icons)
